# 🛒 E-Commerce RESTful Web API

A **backend system** for a full-featured **e-commerce platform**, built using **ASP.NET Core Web API**.
It serves as the **core engine** that handles all business operations behind any online shopping application.

The API is designed to be **secure, scalable, and easy to integrate** with both web and mobile applications.

---

## 🚀 Key Features

* RESTful API design
* Clean Architecture & SOLID principles
* JWT Authentication & Authorization
* Role-based access control
* Electronic payment integration
* Caching for performance optimization
* Global exception handling
* Asynchronous programming for scalability

---

## 🧰 Technologies Used

* ASP.NET Core Web API
* Onion Architecture
* Entity Framework Core
* SQL Server
* JWT Authentication
* AutoMapper
* Generic Repository & Unit of Work
* Specification Pattern
* Redis
